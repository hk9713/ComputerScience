### 연산자 우선순위와 결합방향

|  | 우선순위 | 연산자 | 결합방향 |
| --- | --- | --- | --- |
| 단항연산자 | 1  | () [] → . | → |
| 단항 | 2 | ! ~ ++ -- & * sizeof() cast | ← |
| 이항연산자 | 3 (산술) | * / % | → |
| 이항 | 4 (산술) | + - | → |
| 이항 | 5 | << >> | → |
| 이항 | 6 (관계) | > ≥ < ≤ | → |
| 이항 | 7 (관계) | == ≠ | → |
| 이항 | 8 | & | → |
| 이항 | 9 | ^ | → |
| 이항 | 10 | | | → |
| 이항 | 11 (논리) | && | → |
| 이항 | 12 (논리) | || | → |
| 삼항연산자 | 13 | ? : | ← |
| 삼항 | 14 | = += *= … &= | ← |
| 삼항 | 15 | , | → |

```c
x = 10;
y = ++x;  // 먼저 더해라
printf("x : %d \n", x);  // 11
printf("y : %d \n", y);  // 11

x=10;
y = x++;  // 나중에 더해라
printf("x : %d \n", x);  // 11
printf("y : %d \n", y);  // 10
```

**sizeof()** : 괄호 안에 있는 자료형의 크기 (byte 단위)
